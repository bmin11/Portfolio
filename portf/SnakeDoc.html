Snake on Socket

Intro

For my end of the year project. I wanted to make a web application that could take inputs from multiple users and broadcast the input to all other users. The application itself was not my biggest interest, but I was curious to learn how to directly communicate between users on different browsers without any help of data storage from the server side. I have heard from my friends that Socket.io by far the simplest and most effective framework to use for web socketing and it was based on node.js, which I am fairly familiar with thanks to the Javascript course taken in NSCC.

node.js

Node.js is a asynchronous Javascript runtime. It is essentially a server application written in Javascript. It is known for its scalability as it does not depend on a single thread to handle any connections. Node.js has been one of my favorite learning in NSCC due to its simplicity in implementation and its wide variety in usable libraries, such as Express and Socket.io. Just like Composer for Laravel, most libraries can be downloaded and installed using npm, which stands for node.js Package Manager. The code below has been my go-to for setting up a server for an application, using Express.

var app = require('express')()
var server = require('http').Server(app);

app.get('/', function (req, res) {
res.sendfile(__dirname + '/index.html');
});

server.listen(3000);

Socket.io

As the name implies, Socket.io is a framwork used for web socketting. Socket.io can be initiated by simply adding a single line of require code below.

var io = require('socket.io')( or );

The server starts a connection by listening with the below code

io.on('connection', function (socket) {
//codes
});

With the socket being passed in, the server can now either listen for any incoming sockets

socket.on('disconnect', function () {
io.emit('user disconnected');
});

Or emit a socket to other users to listen to

io.emit('user disconnected');

Any datas, and by mean any datas from string to array of variables, can be attached to the socket by simply adding a parameter in the argument.

io.emit('user disconnected', GoodByeObject);

On the client side, Socket.io can be initiated by simply adding the code below

var socket = io();

And use this just like the server side would listen and broadcast sockets.

There are many other flags that can be added, such as broadcast for emitting the socket to everyone but the sender and volatile for to only receive with good connection. I found the documentation on their website to be sufficient for any questions I had during the development.

Snake

The final product is the snake game played by two players. The player can become the host to start the game and the game is played by using WASD as arrow keys. Each player sends their inputs using Socket.io to the server and the server emits the input to all other players to change the direction of the head. The game emits whenever an orb was eaten to create another orb for the player to chase after. The game finally stops by emitting the victor once the condition is met.
